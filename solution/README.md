# Solution Files

## Clean Version (Recommended)
- `ServiceToolkit_1_0_0_0_clean.zip` — 116 KB, minimal dependencies
- Requires: D365 Customer Service + Product Management (standard with CS license)

## Full Version
- `ServiceToolkit_1_0_0_0.zip` — 4.3 MB, includes Canvas App
- Requires: Field Service, Marketing Email, Sales, Asset Common (not all customers have these)

## Choosing a Version
- **Sending to a customer?** Use the clean version
- **Internal demo env with all apps?** Use the full version
